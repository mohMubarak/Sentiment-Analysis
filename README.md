Sentiment Analysis Using Machine Learning

Overview
This project is an end-to-end Sentiment Analysis application built using Machine Learning 
and Natural Language Processing (NLP) techniques. 
The goal is to classify text into sentiment categories such as Positive, Negative, and Neutral.

we will explore data, do feature engineering, do multiple ML models, and compare their performance.

The project demonstrates the complete machine learning workflow, including:
Data Collection
Data Cleaning and Preprocessing
Text Vectorization
Feature Engineering
Model Training
Model Evaluation
Sentiment Prediction
